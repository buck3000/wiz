Magical git branch contexts for AI coding workflows

Run multiple terminal windows on separate branches at the same time. Each gets its own isolated worktree. Zero confusion, zero conflicts.

Install Free View Pricing 
$ wiz create auth-refactor
# Creates isolated worktree on branch auth-refactor
ctx(auth-refactor) ready at .git/wiz/ctx/auth-refactor

$ wiz create fix-payment-bug
ctx(fix-payment-bug) ready at .git/wiz/ctx/fix-payment-bug

$ wiz spawn auth-refactor
# Opens a new terminal tab, already cd'd into the context

$ wiz ls
  auth-refactor     main + 3 commits
  fix-payment-bug   main + 1 commit
